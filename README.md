# Fortify
A tower defence game in a medieval setting built using Unity

## Prerequisites
- Unity 2018.2.6f1

## Getting Started
1. Clone this repository 
  `git clone https://github.com/stuff-we-deserve/fortify.git`
2. Open with unity.
